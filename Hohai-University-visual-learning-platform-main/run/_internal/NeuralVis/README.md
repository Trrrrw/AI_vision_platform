# NeuralVis - AI算法可视化学习模块

一款零代码、端口交互式神经网络可视化学习工具，支持手动搭建神经网络并实时观察前向传播、反向传播过程。

## 功能特性

- **六大固定功能区**：顶部导航、左侧组件库、中心可视化画布、右侧参数面板、底部结果面板、底部原理讲解区
- **端口交互**：拖拽式端口连接，自动建立组件间数据流
- **网络自定义**：自由指定输入层、隐藏层（层数+每层神经元数）、输出层
- **实时可视化**：动态展示前向传播、反向传播、权重更新
- **任务适配**：分类 / 回归 / 聚类

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行

```bash
python face.py
```
